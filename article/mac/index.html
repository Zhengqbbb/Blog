1. Git

1. 安装Git

  1. 通过homebrew安装Git
  • 安装homebrew
    /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
  • 安装git
    brew install git
  1. 创建ssh key并配置
  • 设置好username喝email，每次commit都会记录（配错不会有小绿点哦）
    git config --global user.name "zhengqbbb"
git config --global user.email "1074059947@qq.com"
  • 创建ssh key
    ssh-keygen -t rsa -C "1074059947@qq.com"

    2. 常见命令

  • 代理
    • 如果自己有github和gitee的项目可以把下面的–global去掉分repo
      # 看看自己的airplane HTTP代理地址是多少
git config --global https.proxy http://127.0.0.1:1080
git config --global https.proxy https://127.0.0.1:1080
# 看看自己的airplane（高级设置） socks5监听地址是多少
git config --global http.proxy 'socks5://127.0.0.1:1080'
git config --global https.proxy 'socks5://127.0.0.1:1080'
git config --global --unset http.proxy
git config --global --unset https.proxy
